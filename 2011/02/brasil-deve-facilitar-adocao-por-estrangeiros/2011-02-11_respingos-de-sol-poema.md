Respingos de sol
================


 


 


  Andros, o Atheu


 



```
Um negro velho varre esta calçada
```


```
Esta calçada varre um negro velho.
```


```
Ele varreu, varreu, tanto varreu,
```


```
Que, como eu, ficou doido varrido.
```


```
 
```


```
Tanto que estalou como arco-íris
```


```
Respingando a ti e a mim de sol...
```
